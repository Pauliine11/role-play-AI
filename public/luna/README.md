# Images de Luna Lovegood

Ce dossier contient les images du personnage **Luna Lovegood** pour le jeu.

## Images requises (format PNG recommandé)

- `neutral.png` - Expression neutre/rêveuse (image par défaut sur la card)
- `happy.png` - Expression joyeuse/émerveillée
- `sad.png` - Expression triste/pensive
- `angry.png` - Expression frustrée (rare pour Luna)
- `desperate.png` - Expression désespérée/perdue

## Spécifications

- **Format** : PNG (avec transparence si possible)
- **Résolution recommandée** : 512x512px minimum
- **Style** : Portrait du visage ou buste
- **Fond** : Transparent ou uniforme pour faciliter l'intégration

## Note d'intégration

Les images utilisent `object-contain` pour préserver les proportions et gérer la transparence PNG.
